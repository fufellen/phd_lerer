# 2D-периодическая решётка Лерера - Fourier Modal Method / RCWA (в разработке)

Независимый (не портированный из кода Лерера) численный солвер для
воспроизведения `R, T, P` двухпериодической решётки диэлектрических
цилиндров с плазмонными наночастицами из статьи [[Лерер АМ_1]] (Obsidian:
`PhD/Scientific_study/Scientific_study/paper_analysis/from_Lerer/от Лерера/
Статья Лерер АМ_1/Лерер АМ_1.md`).

## Почему не порт кода Лерера

Метод самого Лерера (объёмное интегро-дифференциальное уравнение + тензорная
функция Грина слоистой среды через LM/LE-волны + Бессель-базис Галёркина,
статья 2012 г., DOI `10.1134/S106422691210004X`, лежит рядом со статьёй в
Obsidian) частично восстановлен из PDF (часть формул и обозначений в
Приложении потерялась при извлечении текста), и у нас нет референсных чисел
самого Лерера для отладки - воспроизводить его метод вслепую слишком
рискованно.

Вместо этого реализуется **Fourier Modal Method (RCWA)** - независимый,
хорошо задокументированный в литературе метод расчёта `R, T` периодических
слоистых структур с неоднородными (по x,y) слоями. Решает ту же физическую
задачу (дифракция на 2D-периодической решётке из слоёв, часть которых
неоднородна), проверяется на собственных, не связанных с Лерером,
аналитических предельных случаях.

## Цель задачи (что считается успехом)

1. Локализовать резонанс поглощения в пределах ~15-20% по длине волны от
   заявленного в статье (например, "500-600 нм" для золотых ПНЧ).
2. Верно воспроизвести качественные порядки: Au более узкий/резче
   резонанс, чем Cu; TiN более широкополосный, чем HfN (уже подтверждено на
   уровне `composite_ema`, см. `composite_ema/README.md`).
3. Получить `P > 80%` (почти полное поглощение) хотя бы для одной
   конфигурации из статьи - это её центральное заявление.
4. Всё это - только после того, как сам численный метод независимо
   провалидирован (аналитические предельные случаи, сходимость по числу
   учитываемых гармоник Фурье решётки).

Количественное совпадение кривых `R(lambda), T(lambda), P(lambda)` с
графиками статьи - не цель (это потребовало бы точного знания геометрии,
которая в статье не расписана исчерпывающе, и метода Лерера как референса).

## Прогресс

### Стадия 1 - TMM (transfer-matrix method) для однородной многослойной среды - ГОТОВО

`scripts/tmm.py` - расчёт `R, T` (амплитуда и мощность) для стопки
однородных изотропных слоёв, произвольный угол падения, обе поляризации
(рекурсия Парратта). Это одновременно:
- предельный случай RCWA при нулевом числе гармоник решётки (однородный
  слой = решётка без рельефа);
- обязательный строительный блок полной RCWA (сшивка мод между слоями через
  ту же математику согласования импедансов).

`scripts/validate_tmm.py` - независимая валидация против учебных формул:
- формулы Френеля (одна граница, нормальное и наклонное падение, обе
  поляризации) - точное совпадение (`diff < 1e-15`);
- формула Эйри для тонкой плёнки - точное совпадение;
- баланс энергии `R+T=1` для беспотерьного случая по сетке длин волн.

Все проверки проходят. По пути найден и исправлен реальный баг: t-коэффициент
для TM-поляризации был запутан с неверным множителем `n_j/n_j1`, из-за чего
p-поляризация давала T=0.43 вместо 0.96 на простой границе - исправлено на
`t=1+r` (см. комментарии в `tmm.py`).

**Важное найденное несоответствие конвенций.** `composite_ema` хранит `eps`
в конвенции `eps=eps'-i*eps''` (согласовано с исходным C Лерера), а `tmm.py`
написан в стандартной фотонной конвенции `eps=eps'+i*eps''`
(`n=n'+ik, k>0` <=> потери). Это одна и та же физика, разный знак мнимой
части из-за разных временных конвенций (`e^{+iwt}` vs `e^{-iwt}`). При
прямой подстановке `eps_eff` из `composite_ema` в `tmm.py` без учёта этого
получалось нефизичное усиление (`P<0`) вместо поглощения. Исправление:
сопрягать (`.conjugate()`) `eps`/`n` при передаче между модулями - см.
`validate_tmm.py`. Добавлена автоматическая проверка `P >= 0` в демо-прогоне,
чтобы эта ошибка не могла молча вернуться.

Нестрогий (не решётка, план-волна на сплошной плёнке) прогон с
`eps_eff(Au, C=10%)` между подложкой `n=1.45` и воздухом уже показывает
резонансный пик поглощения `P=0.81` на `555 нм` - ровно там, где
[[Условие Фрёлиха (Fröhlich resonance condition)|резонанс Фрёлиха]] был
найден в `composite_ema` (`~540-554 нм`). Это хороший sanity-check самой
машины TMM, но **не результат решётки** - реальная структура периодична и
рассеивает свет в дифракционные порядки, чего однородная плёнка не делает.

### Стадия 2 - Фурье-факторизация решётки и послойные собственные моды - НЕ СДЕЛАНО

Следующий шаг: разложить `eps(x,y)` каждого периодического слоя (диски/
цилиндры) в ряд Фурье по решётке, решить задачу на собственные значения для
мод каждого слоя (с учётом факторизации Ли для разрывной `eps`), сшить слои
через S-матрицу (переиспользуя формализм согласования импедансов из стадии
1), извлечь `R, T` по нулевой дифракционной гармонике.

### Стадия 3 - геометрия статьи - НЕ СДЕЛАНО

Подключить реальную геометрию [[Лерер АМ_1]] (подложка `n=1.45`, слой с ПНЧ
`n=1.77, C=10%, h=100 нм`, цилиндры `n=1.77, C=10%, h=100 нм`, период
решётки) и материалы из `composite_ema` (Au/Cu/HfN/TiN), сравнить с рис. 2-5.

## Запуск

```powershell
python .\grating_2d_rcwa\scripts\validate_tmm.py
```

Требует `numpy`, `matplotlib`. Использует `composite_ema/scripts/
composite_ema.py` напрямую (добавляется в `sys.path`), отдельная установка
не нужна.

Статус: **diagnostic / in progress**. Стадия 1 провалидирована и
энергетически согласована. Стадии 2-3 (собственно решётка и сравнение со
статьёй) не реализованы.
