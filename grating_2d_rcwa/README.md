# 2D-периодическая решётка Лерера - Fourier Modal Method / RCWA (в разработке)

Независимый (не портированный из кода Лерера) численный солвер для
воспроизведения `R, T, P` двухпериодической решётки диэлектрических
цилиндров с плазмонными наночастицами из статьи [[Лерер АМ_1]] (Obsidian:
`PhD/Scientific_study/Scientific_study/paper_analysis/from_Lerer/от Лерера/
Статья Лерер АМ_1/Лерер АМ_1.md`).

## Почему не порт кода Лерера

Метод самого Лерера (объёмное интегро-дифференциальное уравнение + тензорная
функция Грина слоистой среды через LM/LE-волны + Бессель-базис Галёркина,
статья 2012 г., DOI `10.1134/S106422691210004X`, лежит рядом со статьёй в
Obsidian) частично восстановлен из PDF (часть формул и обозначений в
Приложении потерялась при извлечении текста), и у нас нет референсных чисел
самого Лерера для отладки - воспроизводить его метод вслепую слишком
рискованно.

Вместо этого реализуется **Fourier Modal Method (RCWA)** - независимый,
хорошо задокументированный в литературе метод расчёта `R, T` периодических
слоистых структур с неоднородными (по x,y) слоями. Решает ту же физическую
задачу (дифракция на 2D-периодической решётке из слоёв, часть которых
неоднородна), проверяется на собственных, не связанных с Лерером,
аналитических предельных случаях.

## Цель задачи (что считается успехом)

1. Локализовать резонанс поглощения в пределах ~15-20% по длине волны от
   заявленного в статье (например, "500-600 нм" для золотых ПНЧ).
2. Верно воспроизвести качественные порядки: Au более узкий/резче
   резонанс, чем Cu; TiN более широкополосный, чем HfN (уже подтверждено на
   уровне `composite_ema`, см. `composite_ema/README.md`).
3. Получить `P > 80%` (почти полное поглощение) хотя бы для одной
   конфигурации из статьи - это её центральное заявление.
4. Всё это - только после того, как сам численный метод независимо
   провалидирован (аналитические предельные случаи, сходимость по числу
   учитываемых гармоник Фурье решётки).

Количественное совпадение кривых `R(lambda), T(lambda), P(lambda)` с
графиками статьи - не цель (это потребовало бы точного знания геометрии,
которая в статье не расписана исчерпывающе, и метода Лерера как референса).

## Прогресс

### Стадия 1 - TMM (transfer-matrix method) для однородной многослойной среды - ГОТОВО

`scripts/tmm.py` - расчёт `R, T` (амплитуда и мощность) для стопки
однородных изотропных слоёв, произвольный угол падения, обе поляризации
(рекурсия Парратта). Это одновременно:
- предельный случай RCWA при нулевом числе гармоник решётки (однородный
  слой = решётка без рельефа);
- обязательный строительный блок полной RCWA (сшивка мод между слоями через
  ту же математику согласования импедансов).

`scripts/validate_tmm.py` - независимая валидация против учебных формул:
- формулы Френеля (одна граница, нормальное и наклонное падение, обе
  поляризации) - точное совпадение (`diff < 1e-15`);
- формула Эйри для тонкой плёнки - точное совпадение;
- баланс энергии `R+T=1` для беспотерьного случая по сетке длин волн.

Все проверки проходят. По пути найден и исправлен реальный баг: t-коэффициент
для TM-поляризации был запутан с неверным множителем `n_j/n_j1`, из-за чего
p-поляризация давала T=0.43 вместо 0.96 на простой границе - исправлено на
`t=1+r` (см. комментарии в `tmm.py`).

**Важное найденное несоответствие конвенций.** `composite_ema` хранит `eps`
в конвенции `eps=eps'-i*eps''` (согласовано с исходным C Лерера), а `tmm.py`
написан в стандартной фотонной конвенции `eps=eps'+i*eps''`
(`n=n'+ik, k>0` <=> потери). Это одна и та же физика, разный знак мнимой
части из-за разных временных конвенций (`e^{+iwt}` vs `e^{-iwt}`). При
прямой подстановке `eps_eff` из `composite_ema` в `tmm.py` без учёта этого
получалось нефизичное усиление (`P<0`) вместо поглощения. Исправление:
сопрягать (`.conjugate()`) `eps`/`n` при передаче между модулями - см.
`validate_tmm.py`. Добавлена автоматическая проверка `P >= 0` в демо-прогоне,
чтобы эта ошибка не могла молча вернуться.

Нестрогий (не решётка, план-волна на сплошной плёнке) прогон с
`eps_eff(Au, C=10%)` между подложкой `n=1.45` и воздухом уже показывает
резонансный пик поглощения `P=0.81` на `555 нм` - ровно там, где
[[Условие Фрёлиха (Fröhlich resonance condition)|резонанс Фрёлиха]] был
найден в `composite_ema` (`~540-554 нм`). Это хороший sanity-check самой
машины TMM, но **не результат решётки** - реальная структура периодична и
рассеивает свет в дифракционные порядки, чего однородная плёнка не делает.

### Стадия 2 - Фурье-факторизация решётки и послойные собственные моды (1D) - ГОТОВО

`scripts/rcwa_1d_te.py` - RCWA для **1D-периодической** решётки (полоски,
период только по x; см. "Упрощение геометрии" ниже), TE-поляризация.
Фурье-коэффициенты полоски (замкнутая формула через `sin(pi*i*f)/(pi*i)`),
Тёплицева матрица свёртки, собственные моды слоя (`Kx@Kx - eps_toeplitz`,
ветвь `Im(q)>=0`), матричное обобщение интерфейс/пропагационных матриц
`tmm.py` на многогармонический случай (полный вывод - в докстринге файла).

Первая попытка использовала "компактную" gap-medium формулу RCWA из
учебных заметок и **не сошлась** с уже провалидированным `tmm.py` даже в
тривиальном однослойном пределе (значит, формула была неверно
воспроизведена по памяти) - переделано на прямое матричное обобщение
собственных, уже проверенных формул `tmm.py`, вручную сверенное со
скалярным случаем перед реализацией (см. докстринг `rcwa_1d_te.py`).

`scripts/validate_rcwa_1d.py` - независимая валидация:
- Фурье DC-член = средневзвешенная eps (`f*eps_incl+(1-f)*eps_bg`);
- собственные моды однородного слоя (без решётки) точно совпадают с прямым
  расчётом `kz` по гармоникам;
- решётка с `fill=0` (нет неоднородности) **точно** совпадает с `tmm.py`
  для того же однородного слоя (`diff < 1e-6`), и вся мощность остаётся в
  нулевой гармонике (никакой паразитной дифракции без реальной решётки);
- баланс энергии `sum(R)+sum(T)=1` для беспотерьной решётки по сетке длин
  волн (`|error| < 1e-6`);
- сходимость по числу гармоник `N` (`R_0` сходится: `N=2:0.384, N=4:0.361,
  N=6:0.3613, N=9:0.3613`).

Все проверки проходят.

### Стадия 3 - геометрия рис. 2 статьи (Au vs Cu) - ПЕРВЫЙ РЕЗУЛЬТАТ ЕСТЬ

`scripts/reproduce_fig2_au_vs_cu.py` - геометрия из подписи к рис. 2 и
текста "Результаты" статьи: подложка `n=1.45`, диэлектрический слой с ПНЧ
`h=100 нм, n=1.77, C=10%`, цилиндры `h=100 нм, n=1.77, C=10%`, период
`500 нм`, диаметр `400 нм` (заменены на 1D-полоски той же заполняющей
доли `f=diameter/period=0.8` - см. "Упрощение геометрии"), материалы Au и
Cu из `composite_ema`.

**Результат по цели задачи** (см. выше):
- Резонанс: Au на **525 нм**, Cu на **517 нм** (край доступного диапазона
  таблицы Cu) - оба в заявленном статьёй диапазоне 500-600 нм, близко к
  условию Фрёлиха из `composite_ema` (Au ~540 нм, Cu ~528 нм). **Критерий 1
  выполнен.**
- `P_max`: Au = **0.877**, Cu = **0.776** - Au даёт более глубокий пик,
  качественно как и ожидалось (Au - более резкий/глубокий резонанс, Cu
  гасится межзонным поглощением, см. `composite_ema/README.md`).
  **Критерий 3 (`P>0.8`) выполнен для Au.**
- Энергия сохраняется точно (`R+T+P=1` по построению, потерянная мощность
  = `P`, никакого "утекания" в непросчитанные каналы).
- Метод независимо провалидирован ДО этого прогона (стадия 2 выше).
  **Критерий 4 выполнен.**
- Найден дополнительный узкий резонанс `P≈0.44` на `~855 нм` - устойчив
  при увеличении `N` (6/10/14 дают одинаковый результат до 3-го знака),
  то есть не численный артефакт, а особенность модели (вероятно,
  guided-mode/Fano-резонанс от решётки, специфичный для 1D-приближения -
  не пытаться напрямую сопоставлять с рис. 2 статьи без дополнительной
  проверки).

Результаты: `results/reproduce_fig2_au_vs_cu.png`, `.csv`.

**Критерий 2 (Au vs Cu УФ-сравнение из рис. 2) по-прежнему не проверен** -
у таблицы Cu в `composite_ema` нет данных короче 517 нм (см.
`composite_ema/README.md`), тот же изначальный пробел.

### Стадия 3, рис. 5 (HfN vs TiN) - критерий 2 подтверждён на уровне полной решётки

`scripts/reproduce_fig5_hfn_vs_tin.py` - та же геометрия, материалы HfN и
TiN вместо Au/Cu. Результат:

- `P_max`: HfN = **0.820** (>0.8, критерий 3 выполнен и для HfN), TiN =
  **0.729**.
- Полоса `P >= half-max`: **HfN 440 нм** (375-815) против **TiN 535 нм**
  (350-885) - **TiN шире HfN**, что качественно совпадает со статьёй
  ("Заключение": "ДР с плазмонными наночастицами из TiN более
  широкополосные"). Это тот же качественный вывод, что уже был получен на
  уровне одной формулы (1) без решётки (`composite_ema`, полоса 346 vs
  208 нм) - теперь подтверждён и после добавления настоящей дифракции на
  решётке, не только для однородной эффективной среды.

Результаты: `results/reproduce_fig5_hfn_vs_tin.png`, `.csv`.

**Итог по цели задачи (все 4 пункта):** 1 - выполнен (резонанс в
диапазоне, близко к условию Фрёлиха); 2 - выполнен (TiN шире HfN,
подтверждено дважды: `composite_ema` и полная решётка); 3 - выполнен (Au
`P=0.877`, HfN `P=0.820`, оба `>0.8`); 4 - выполнен (стадии 1-2
провалидированы независимо до стадии 3). Единственный оставшийся пробел -
УФ-сравнение Au/Cu (нет УФ-данных Cu, известно с самого начала работы).

### Стадия 4 - НАСТОЯЩАЯ 2D-решётка цилиндров (crossed grating) - ГОТОВО

`scripts/rcwa_2d.py` - полновекторная FMM для би-периодической решётки
(квадратная решётка, нормальное падение). Полный вывод блочных матриц
`F`, `G` (через исключение `Ez`, `Hz` из уравнений Максвелла в
Фурье-пространстве) - в докстринге файла; конвенции (временной фактор,
ветвь `Im(q)>=0`, интерфейсные/пропагационные матрицы) те же, что в уже
провалидированных `tmm.py`/`rcwa_1d_te.py`. Фурье-коэффициенты диска -
аналитика через `J1` (локальная аппроксимация A&S, scipy не нужен).
Факторизация eps - правило Лорана (без инверсных правил Ли): для
умеренного контраста композит/воздух сходимость подтверждена явно
(P стабилен до 4-го знака уже при `N=4`, см. валидацию). Потоки мощности
считаются из полных поперечных полей гармоник (`Sz ~ Re(ex hy* - ey hx*)`),
а не из `|c|^2` - на косых порядках (m,n) поляризационные базисные моды
дают перекрёстные члены.

`scripts/validate_rcwa_2d.py` - валидация ДО сравнения со статьёй:
- `J1` против ряда Тейлора; Фурье диска (DC = доля площади, симметрии);
- однородный предел == `tmm.py` (`diff < 1e-6`), вся мощность в (0,0);
- **1D-предел (профиль-полоска) == `rcwa_1d_te.py` по каждому порядку
  (`diff ~ 1e-15`)** - самый сильный тест всей векторной машины;
- баланс энергии для беспотерьных цилиндров (`|err| < 1.5e-10`);
- симметрия x-pol == y-pol для цилиндров на квадратной решётке (`~1e-16`);
- сходимость по `N` (P: N=2: 0.3333, N=3: 0.3267, N=4: 0.3231, N=5: 0.3218).

`scripts/reproduce_fig2_au_vs_cu_2d.py` - та же геометрия рис. 2 статьи,
но с настоящими цилиндрами (доля площади `pi*(D/2)^2/L^2 = 0.503` вместо
1D-шной `D/L = 0.8`), `N=5` (121 гармоника). Результаты
(`results/reproduce_fig2_au_vs_cu_2d.png`, `.csv`):

- Резонанс Au: **535 нм**, `P_max = 0.925` - ближе к условию Фрёлиха
  (~540 нм) и глубже, чем в 1D (525 нм, 0.877); согласуется с центральным
  заявлением статьи про поглощение >90% в видимом диапазоне.
- Cu: глобальный максимум в УФ (215 нм, `P=0.847`), в видимом - широкое
  плато `P~0.80` около 500-535 нм.
- Вместо одного узкого пика 855 нм (1D) - два узких резонанса ~770 и
  ~835 нм (guided-mode/Fano от двух семейств порядков 2D-решётки).
- **УФ-сравнение Au/Cu: расхождение со статьёй НЕ снимается 2D-геометрией.**
  Среднее `P` в УФ (<=400 нм): Au 0.705, Cu 0.768 - Cu по-прежнему сильнее
  Au, как и в 1D (0.770/0.828), хотя статья утверждает обратное. Гипотеза
  "виновато 1D-приближение" исключена; остаётся главная гипотеза -
  разные первоисточники оптических констант (luxpop у Лерера против
  Johnson & Christy 1972 здесь).

## Упрощение геометрии (историческая оговорка для стадии 3)

Стадия 3 использовала **1D-полоски** вместо реальных 2D-цилиндров статьи.
Причина: 1D-случай при внутриплоскостном падении точно расцепляет TE/TM
(не нужны crossed-grating уравнения), что снижало риск незаметной ошибки
в знаках/индексах - уже один раз пришлось переделывать формализм из-за
ошибки в формуле, взятой "по памяти" (см. выше). Со стадии 4 это
ограничение снято: `rcwa_2d.py` считает настоящие цилиндры, а 1D-код
остался как независимый предельный случай для валидации 2D-кода.

## Запуск

```powershell
python .\grating_2d_rcwa\scripts\validate_tmm.py
python .\grating_2d_rcwa\scripts\validate_rcwa_1d.py
python .\grating_2d_rcwa\scripts\validate_rcwa_2d.py
python .\grating_2d_rcwa\scripts\reproduce_fig2_au_vs_cu.py
python .\grating_2d_rcwa\scripts\reproduce_fig2_au_vs_cu_2d.py   # ~7 мин
```

Требует `numpy`, `matplotlib` (scipy НЕ нужен - J1 реализован локально).
Использует `composite_ema/scripts/composite_ema.py` напрямую (добавляется
в `sys.path`), отдельная установка не нужна.

Статус: **validated (метод) / diagnostic (сравнение со статьёй)**.
Стадии 1, 2, 4 провалидированы независимо от статей Лерера (Френель/Эйри,
взаимные пределы, энергия, сходимость, симметрия). Стадия 3 (1D) и
стадия 4 (2D цилиндры) воспроизводят рис. 2/5 статьи качественно:
резонанс на месте, `P>80%` (в 2D `P=0.925>90%`), TiN шире HfN;
УФ-сравнение Au/Cu расходится со статьёй в обеих геометриях - открытый
вопрос про первоисточник оптических констант. Рис. 3-4 статьи
(зависимость от периода/диаметра) - не проверены.
